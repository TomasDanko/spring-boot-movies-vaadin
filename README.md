Webová aplikácia na správu filmov postavená na **Spring Boot** a **Vaadin Flow**.  
Projekt slúži ako ukážka prepojenia backendovej logiky (JPA, služby) s moderným server-side UI vo Vaadine.

---

## Použité technológie

- Java 17
- Spring Boot 3
- Spring Data JPA
- PostgreSQL
- Vaadin 24
- Maven
- Git & GitHub
